# @ifiokjr/oh-pi

> All-in-one pi package: extensions, themes, prompts, skills, and ant-colony swarm.

```bash
pi install npm:@ifiokjr/oh-pi
```

This meta-package bundles all oh-pi resources. Install individual packages if you only need specific
features:

| Package                 | Install                                | Contents                                                                                    |
| ----------------------- | -------------------------------------- | ------------------------------------------------------------------------------------------- |
| `@ifiokjr/oh-pi-extensions` | `pi install npm:@ifiokjr/oh-pi-extensions` | safe-guard, git-guard, auto-session, custom-footer, compact-header, auto-update, bg-process |
| `@ifiokjr/oh-pi-ant-colony` | `pi install npm:@ifiokjr/oh-pi-ant-colony` | Multi-agent swarm extension                                                                 |
| `@ifiokjr/oh-pi-themes`     | `pi install npm:@ifiokjr/oh-pi-themes`     | cyberpunk, nord, gruvbox, tokyo-night, catppuccin, oh-p-dark                                |
| `@ifiokjr/oh-pi-prompts`    | `pi install npm:@ifiokjr/oh-pi-prompts`    | review, fix, explain, refactor, test, commit, pr, and more                                  |
| `@ifiokjr/oh-pi-skills`     | `pi install npm:@ifiokjr/oh-pi-skills`     | web-search, debug-helper, git-workflow, and more                                            |

## TUI Configurator

```bash
npx @ifiokjr/oh-pi-cli
```
